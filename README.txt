# Multilayer-Jointly-Evolving-and-Compressing-Fuzzy-Neural-Network

The MATLAB code for the research paper titled "Multilayer Evolving Fuzzy Neural Networks
with Self-Adaptive Dimensionality Compression for High-Dimensional Data Classification".

This work is described in:
=========================================================================================
Gu, X., Ni, Q., Shen, Q. (2024). Multilayer Evolving Fuzzy Neural Networks with 
Self-Adaptive Dimensionality Compression for High-Dimensional Data
Classification. IEEE Transactions on Fuzzy Systems, accepted.
=========================================================================================
Please cite the paper above if this code helps. 
Programmed by Xiaowei Gu.
For any queries about the code, please contact Dr. Xiaowei Gu: xiaowei.gu@surrey.ac.uk
Note, the dataset in this repository is a subset of MNIST dataset.
